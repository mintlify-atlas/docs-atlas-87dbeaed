<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">

    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with VitaFem Backend in minutes
  </p>
  
  <Steps>
    <Step title="Set up your environment">
      Configure your Laravel environment with Oracle database credentials. Create a `.env` file with your database connection settings.
      
      ```bash
      DB_CONNECTION=oracle
      DB_HOST=your-oracle-host
      DB_PORT=1521
      DB_DATABASE=your-database
      DB_USERNAME=your-username
      DB_PASSWORD=your-password
      ```
    </Step>
    
    <Step title="Register a user account">
      Create a new user account via the registration endpoint to get started.
      
      ```bash cURL
      curl -X POST https://your-api-domain.com/api/register \
        -H "Content-Type: application/json" \
        -d '{
          "name": "Dr. Maria Garcia",
          "email": "maria@vitafem.com",
          "password": "securePassword123"
        }'
      ```
    </Step>
    
    <Step title="Authenticate and get your token">
      Log in with your credentials to receive an authentication token for API access.
      
      ```bash cURL
      curl -X POST https://your-api-domain.com/api/login \
        -H "Content-Type: application/json" \
        -d '{
          "email": "maria@vitafem.com",
          "password": "securePassword123"
        }'
      ```
      
      <Accordion title="Example response">
        ```json
        {
          "message": "Login exitoso",
          "user": {
            "id_usuario": 1,
            "nombre": "Dr. Maria Garcia",
            "correo": "maria@vitafem.com",
            "activo": "1"
          },
          "token": "1|abc123def456..."
        }
        ```
      </Accordion>
    </Step>
    
    <Step title="Make your first API call">
      Use your authentication token to access protected endpoints like fetching the list of doctors.
      
      ```bash cURL
      curl -X GET https://your-api-domain.com/api/medicos \
        -H "Authorization: Bearer YOUR_TOKEN_HERE"
      ```
    </Step>
  </Steps>
</div>

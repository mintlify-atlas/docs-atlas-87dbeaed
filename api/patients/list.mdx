## Implementation Details

### Oracle Database

This endpoint queries the `PACIENTES` table in Oracle Database:

- **Table**: `PACIENTES`
- **Primary Key**: `ID_PACIENTE` (auto-incremented via `SEQ_PACIENTES` sequence)
- **Timestamps**: Disabled (`public $timestamps = false`)

### Controller Method

Implemented in `PacienteController::index()` at `app/Http/Controllers/PacienteController.php:15`

```php
public function index()
{
    // SELECT * FROM PACIENTES
    $pacientes = Paciente::all();
    return response()->json($pacientes);
}
```

## Notes

<Warning>
  **Field Name Casing**: Oracle column names are defined in UPPERCASE in the model, but the Yajra Oracle driver may return them in lowercase when reading. Be prepared to handle both cases in your client application.
</Warning>

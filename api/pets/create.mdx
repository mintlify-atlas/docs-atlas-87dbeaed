---
title: "Create Pet"
api: "POST /api/pets"
description: "Create a new pet for the authenticated user"
---

## Authentication

This endpoint requires authentication using Laravel Sanctum. Include the bearer token in the Authorization header.

```bash
Authorization: Bearer {token}
```

## Request Body

<ParamField body="name" type="string" required>
  Name of the pet
</ParamField>

<ParamField body="owner_name" type="string" required>
  Name of the pet's owner
</ParamField>

<ParamField body="age" type="integer" required>
  Age of the pet in years
</ParamField>

<ParamField body="breed" type="string" required>
  Breed of the pet
</ParamField>

<ParamField body="photo" type="file" optional>
  Photo of the pet. Accepts jpeg, png, jpg, gif formats. Maximum size: 2MB.
</ParamField>

## Response

Returns the newly created pet object.

<ResponseField name="id" type="integer">
  Unique identifier for the pet
</ResponseField>

<ResponseField name="name" type="string">
  Name of the pet
</ResponseField>

<ResponseField name="owner_name" type="string">
  Name of the pet's owner
</ResponseField>

<ResponseField name="age" type="integer">
  Age of the pet in years
</ResponseField>

<ResponseField name="breed" type="string">
  Breed of the pet
</ResponseField>

<ResponseField name="photo_url" type="string">
  URL to the pet's photo
</ResponseField>

<ResponseField name="user_id" type="integer">
  ID of the user who owns this pet
</ResponseField>

## Example Request

```bash
curl -X POST https://api.vitafem.com/api/pets \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Accept: application/json" \
  -F "name=Max" \
  -F "owner_name=John Doe" \
  -F "age=3" \
  -F "breed=Golden Retriever" \
  -F "photo=@/path/to/photo.jpg"
```

## Example Response

```json
{
  "id": 1,
  "name": "Max",
  "owner_name": "John Doe",
  "age": 3,
  "breed": "Golden Retriever",
  "photo_url": "https://api.vitafem.com/storage/pets/photo.jpg",
  "user_id": 5
}
```

---
title: "List Products"
api: "GET /api/productos"
description: "Retrieve all products available in the catalog"
---

## Authentication

This endpoint is public and does not require authentication.

## Response

Returns an array of product objects with public image URLs.

<ResponseField name="products" type="array">
  Array of product objects
  
  <ResponseField name="id" type="integer">
    Unique identifier for the product
  </ResponseField>
  
  <ResponseField name="nombre" type="string">
    Name of the product
  </ResponseField>
  
  <ResponseField name="descripcion" type="string">
    Description of the product
  </ResponseField>
  
  <ResponseField name="precio" type="number">
    Price of the product
  </ResponseField>
  
  <ResponseField name="stock" type="integer">
    Available stock quantity
  </ResponseField>
  
  <ResponseField name="imagen" type="string">
    Public URL to the product image
  </ResponseField>
  
  <ResponseField name="created_at" type="string">
    Timestamp when the product was created
  </ResponseField>
  
  <ResponseField name="updated_at" type="string">
    Timestamp when the product was last updated
  </ResponseField>
</ResponseField>

## Example Request

```bash
curl -X GET https://api.vitafem.com/api/productos \
  -H "Accept: application/json"
```

---
title: "Delete User"
api: "DELETE /api/users/{id}"
description: "Delete a user from the system"
---

## Authentication

This endpoint requires authentication using Laravel Sanctum. Include the bearer token in the Authorization header.

```bash
Authorization: Bearer {token}
```

## Path Parameters

<ParamField path="id" type="integer" required>
  The ID of the user to delete
</ParamField>

## Response

Returns a success message confirming the deletion.

<ResponseField name="message" type="string">
  Confirmation message
</ResponseField>

## Example Request

```bash
curl -X DELETE https://api.vitafem.com/api/users/1 \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Accept: application/json"
```

## Example Response

```json
{
  "message": "Eliminado"
}
```

## Notes

- This operation is permanent and cannot be undone
- Returns 404 if user not found
- Consider the implications of deleting users with associated data (pets, appointments, etc.)
